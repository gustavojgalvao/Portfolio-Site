Fried Chicken

Site fictício focado em experiência imersiva através de animações CSS e JavaScript.

HTML5 CSS Animations JavaScript

O Contexto & Desafio

A apresentação de produtos alimentícios frequentemente peca pela estaticidade nas interfaces. O desafio consistia em simular uma marca fictícia de "Frango Frito" elevando a apresentação do produto a um nível altamente tátil. A página deveria "da fome" instantaneamente, apelando para animações orgânicas, cores ricas (food styling digital) e navegação responsiva e interativa.

A Solução Técnica

Desenvolvi uma landing page que abusa de Keyframes do CSS para entradas de elementos e JavaScript para acoplar movimento de parallax ao scroll do usuário. O projeto é responsivo do núcleo, utilizando grids semânticos e Flexbox refinados. A paleta de cores foca no contraste emocional para a gastronomia (vermelhos e amarelos vibrantes) contrapondo-se com tipografia de forte personalidade.

Aprendizados & Impacto

Através da implementação manual de interações complexas neste projeto, refinei incrivelmente minha capacidade de manipular os vetores de transição CSS garantindo 60fps constantes até mesmo em telefones da geração anterior.

Tecnologias

  • HTML Semantic Layer
  • Advanced CSS Keyframes
  • Vanilla JavaScript
  • Vercel Deployment
