Hestus AI

Landing page corporativa para uma empresa de automação B2B movida por Inteligência Artificial.

HTML CSS UI/UX B2B

O Contexto & Desafio

Vender serviços de IA B2B (Business-to-Business) demanda uma percepção cristalina de confiança, alta tecnologia corporativa e clareza nos retornos (ROI). O desafio com a "Hestus" era criar uma fundação visual impecável e futurística que transicionasse rapidamente um Lead frio (pequenos e médios negócios) para um visitante convencido de que aplicar IA era o único caminho racional.

A Solução Técnica

Adotei uma estrutura limpa onde o conteúdo é o absoluto protagonista. Espaços negativos massivos em torno das Proposições de Valor (USP) garantem leitura livre de carga cognitiva. Desenvolvi transições suaves de conteúdo que engajam psicologicamente sem distrair o olhar da meta primária: captura do lead.

Aprendizados & Impacto

Foi uma verdadeira consolidação em "Design para Conversão B2B", aprendendo como microinterações focadas transmitem autoridade corporativa massiva, mesmo em escopos HTML e CSS relativamente simples. O refinamento contínuo das hierarquias textuais foi a maior vitória da iteração.

Tecnologias

  • HTML5 Semântico
  • Modular CSS
  • UX Conversion Focus
  • Vercel Deployment
