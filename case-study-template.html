BRL2GO

Conversor de moedas moderno, rápido e profissional com taxas de câmbio em tempo real.

JavaScript CSS UI/UX API Integration

O Desafio

Desenvolver uma plataforma de conversão de moedas rápida, confiável e com uma interface premium, garantindo que usuários finais e profissionais possam obter cotações precisas instantaneamente sem complexidade desnecessária. A experiência precisava ser tão fluida quanto um aplicativo nativo.

A Solução

A aplicação foi estruturada usando Vanilla JavaScript para máxima performance, integrando com APIs de taxas de câmbio globais. A interface prioriza o minimalismo e feedback imediato através de microinterações. Utilizei um design system próprio para criar os tokens visuais.

Aprendizados & Impacto

Este projeto aprofundou meus conhecimentos no ecossistema de APIs REST, tratamento assíncrono avançado com Promises/Async Await, gerenciamento de estado puramente em front-end, e otimização de renderização do DOM em dispositivos móveis.

Tecnologias

  • HTML5 / Modern CSS3
  • ES6+ JavaScript
  • ExchangeRate-API
  • Git / Vercel
